Arquitetura & Engenharia

Desenvolvido com foco em alta disponibilidade, inteligência artificial preditiva e segurança militar de redes.

🌐 Django & ASGI Daphne

Nossa base robusta em Python Django lida com transações financeiras críticas, faturamentos e persistência, enquanto o canal assíncrono ASGI via Daphne processa WebSockets para telemetria satelital em tempo real.

📍 PostgreSQL & PostGIS

Armazenamento de dados geográficos e cálculo geométrico de distâncias e áreas de risco operacional rodoviário. Permite a busca ultraveloz por motoristas disponíveis em raios de atuação geográfica.

⚡ Redis Caching & Memory

Gerenciamento de filas de telemetria GPS e cache distribuído de alta velocidade. Absorve a carga contínua de envios de coordenadas telemétricas sem onerar a base relacional PostgreSQL.

Visualização da Topologia de Rede

A infraestrutura FreteHub é isolada em uma VPC segura na nuvem, dividida em sub-redes estratégicas públicas e privadas, garantindo isolamento total dos bancos de dados.

[Usuário Mobile / Web] │ ▼ [DNS Gerenciado] ──► [CDN + WAF] ──► [Load Balancer] ──► [API Gateway] │ ┌──────────────────────────────────────────────────────────┘ ▼ [VPC / Rede Privada FreteHub] ├── Sub-rede Pública: │ └── NAT Gateway (Saída de dados controlada de APIs) │ ├── Sub-rede Privada de Aplicação: │ ├── Django ASGI Server (Telemetria) │ ├── Django WSGI Web Server (Painéis operacionais) │ └── Motor RAG IA & Embeddings │ └── Sub-rede Privada de Dados (Isolada da Internet): ├── PostgreSQL + PostGIS (Estruturado e Geográfico) └── Redis Cache (Posição de frotas e telemetria volátil)
Sub-rede Privada de Dados

Todos os bancos de dados estão alocados em sub-redes isoladas, sem IP público ou acesso direto de entrada pela internet. Toda comunicação é exclusivamente autorizada para servidores da aplicação.

Integrações de Saída Seguras

Requisições para APIs de mapas (Google Maps), serviços bancários (Pix) ou notas fiscais (SEFAZ) passam obrigatoriamente por firewalls de saída controlados e gateways de tradução de endereços (NAT).
