# Conectividade — FreteHub

## APIs e Endpoints Principais

### Autenticação (accounts)
| Método | Endpoint | Descrição |
|--------|----------|-----------|
| POST | `/api/auth/register/` | Autocadastro |
| POST | `/api/auth/login/` | Login JWT |
| POST | `/api/auth/refresh/` | Refresh token |
| POST | `/api/auth/logout/` | Logout (blacklist) |
| GET | `/api/auth/me/` | Dados do usuário |
| GET | `/api/users/me/` | Perfil detalhado |
| PATCH | `/api/users/me/` | Atualizar perfil |

### Empresas e Transportadoras (companies)
| Método | Endpoint | Descrição |
|--------|----------|-----------|
| POST | `/api/companies/` | Criar perfil PJ |
| GET | `/api/companies/me/` | Ver perfil PJ |
| PATCH | `/api/companies/{id}/` | Atualizar PJ |
| POST | `/api/carriers/` | Criar transportadora |
| GET | `/api/carriers/me/` | Ver transportadora |
| PATCH | `/api/carriers/{id}/` | Atualizar transportadora |

### Logs de Auditoria (audit)
| Método | Endpoint | Descrição |
|--------|----------|-----------|
| GET | `/api/audit-logs/` | Listar logs |
| GET | `/api/audit-logs/{id}/` | Ver log específico |

## WebSocket (Django Channels)

### Configuração ASGI
```
ASGI_APPLICATION = 'config.asgi.application'
```

### Channel Layers
```python
CHANNEL_LAYERS = {
    'default': {
        'BACKEND': 'channels.layers.InMemoryChannelLayer',
    }
}
```
Modo in-memory (desenvolvimento). Em produção: Redis.

### Possíveis Canais WebSocket
- `tracking/` — Atualizações de localização em tempo real
- `notifications/` — Notificações push
- `chat/` — Comunicação com assistente IA
- `freight_updates/` — Mudanças de status de frete
